
# Aufgabe: Streams

Gegeben sei ein Array mit unsortierten Zahlen:
int[] zahlen = new int[]{9, 1, 8, 2, 7, 3, 6, 4, 5};

- Sortiere die Liste mit Arrays.stream(zahlen)
- Berechne die Summe der Zahlen.
- Berechne das Produkt aller Zahlen.
